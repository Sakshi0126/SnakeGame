# Snake Game in C++

A classic Snake Game implemented in C++ using Object-Oriented Programming, multithreading, and keyboard event handling. This project is designed for learning and enjoyment, with a modular architecture.

## Features
- OOP-based design (Snake, Board, Game classes)
- Multithreading for game loop and input
- Keyboard event handling (conio.h)
- Timers for movement and events
- Level progression and scoring

## Build Instructions
1. Install a C++ compiler (e.g., g++).
2. Build using the provided tasks or run:
   ```
   g++ -std=c++17 -pthread src/*.cpp -o SnakeGame.exe
   ```
3. Run the game:
   ```
   SnakeGame.exe
   ```

## Controls
- Arrow keys: Move snake
- 'q': Quit game
